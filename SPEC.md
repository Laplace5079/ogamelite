# OGame Lite - 增强版规格说明书

## 🎯 核心差异 (对比原版 OGame)

### 1. AI 对手系统
- **单人战役模式** - 对抗 AI 玩家
- **AI 发展策略** - 进攻型/防守型/经济型
- **动态宇宙** - AI 玩家会建设、攻击、殖民

### 2. 改进的建筑系统
- 建筑升级队列可视化
- 建筑时间预览
- 一键升级优化

### 3. 战斗系统增强
- 3D 战斗动画
- 详细的战斗报告分析
- 阵型选择

### 4. 经济系统
- 资源交易系统
- 市场/拍卖行
- 每日任务奖励

### 5. 进度系统
- 成就系统
- 每日/每周挑战
- 排行榜 (vs AI)

### 6. 用户体验
- 更好的 UI/UX
- 快捷键支持
- 资源预警
- 建设建议

## AI 系统设计

### AI 玩家类型
```typescript
enum AIDifficulty {
  EASY = 1,      // 休闲 AI
  NORMAL = 2,    // 普通 AI  
  HARD = 3,      // 困难 AI
  INSANE = 4     // 疯狂 AI
}

enum AIStrategy {
  ECONOMIC,      // 经济发展
  MILITARY,      // 军事扩张
  BALANCED       // 平衡发展
}

interface AIPlayer {
  id: string;
  name: string;
  difficulty: AIDifficulty;
  strategy: AIStrategy;
  personality: {
    aggression: number;      // 0-1 攻击性
    expansion: number;       // 0-1 扩张欲
    defense: number;          // 0-1 防御意识
  };
}
```

### AI 决策循环
1. 资源评估 - 检查资源充足度
2. 建筑优先级 - 根据策略决定建设
3. 舰队生产 - 积累舰队力量
4. 侦察分析 - 扫描目标星球
5. 行动决策 - 攻击/防御/殖民/贸易

## 战役模式

### 教程关卡
1. 基础建设 - 学习资源建筑
2. 首次殖民 - 解锁殖民船
3. 舰队初建 - 建造第一支舰队
4. 首次战斗 - 攻击海盗
5. 联盟初体验 - 了解社交

### 对抗 AI 关卡
- 第1关: 弱小的 AI 邻居
- 第5关: 经济型 AI
- 第10关: 军事型 AI
- 第15关: 超级 AI 大 boss
